# Российский рынок строительных тендеров: полный технический обзор

Российская экосистема строительных тендеров представляет собой сложную, многоуровневую инфраструктуру из **8 федеральных электронных торговых площадок**, десятков коммерческих платформ, государственных сметных баз и растущего числа аналитических сервисов. Центральным звеном остаётся ЕИС (zakupki.gov.ru), однако **с 1 января 2025 года FTP-сервер ЕИС закрыт** — массовая выгрузка данных теперь требует авторизации через SOAP API. Строительный сегмент — крупнейший по объёму госзакупок, но при этом самый технически сложный для автоматизации из-за преобладания сканированных PDF и отсутствия стандартизации документации. Критически важный вывод для разработчиков: **автоматическая генерация сметных прототипов через AI — полностью незанятая ниша**, ни один массовый сервис на российском рынке этого не предлагает.

## 1. Федеральные ЭТП: восемь площадок и один агрегатор

В России аккредитовано **8 федеральных электронных торговых площадок** для проведения государственных закупок по 44-ФЗ и 223-ФЗ (Распоряжение Правительства РФ №1447-р от 12.07.2018). Все они интегрированы с Единой информационной системой (ЕИС).

**ЕИС (zakupki.gov.ru)** — центральный портал размещения информации обо всех госзакупках. Оператор — Федеральное казначейство. Все 8 ЭТП обязаны передавать данные в ЕИС, что делает его единственным полным агрегатором. Поиск тендеров открыт без регистрации, подача заявок — только через ЭТП с квалифицированной электронной подписью (КЭП). Строительные тендеры фильтруются по ОКПД2 (коды 41.2, 42, 43).

**Сбербанк-АСТ (sberbank-ast.ru)** — безусловный лидер рынка с долей **более 40%** объёма торгов по 44-ФЗ. Через универсальную торговую площадку (УТП) обслуживает крупнейших заказчиков строительных работ, включая Росатом и Транснефть. Публичного API нет, данные передаются в ЕИС.

**РТС-тендер (rts-tender.ru)** занимает **около 25%** рынка и имеет специализированную секцию по ПП РФ №615 для капитального ремонта (615.rts-tender.ru), что особенно важно для строительных компаний. **Росэлторг (roseltorg.ru)** — третья по объёму площадка с профильными секциями для капремонта (fkr2.roseltorg.ru) и корпоративных закупок «Россетей» и «Ростеха».

Остальные федеральные площадки — **ЗаказРФ** (zakazrf.ru, ~1% рынка), **НЭП/Фабрикант** (etp-ets.ru, ~6%, обслуживает Росатом, ОСК, Роскосмос), **ЭТП ГПБ** (etpgpb.ru, фокус на Газпром), **ТЭК-Торг** (tektorg.ru, нефтегазовый сектор) и **АСТ ГОЗ** (astgoz.ru, закрытые торги по гособоронзаказу — полностью недоступны для внешнего мониторинга, требуется спецПО VipNet/Континент АП).

**Общее правило доступа:** просмотр опубликованных закупок на всех федеральных ЭТП открыт без регистрации. Для участия в торгах обязательны регистрация в ЕИС через Госуслуги, КЭП и спецсчёт.

## 2. Коммерческие тендеры и агрегаторы данных

Коммерческие тендеры (вне 44-ФЗ и 223-ФЗ) не попадают в ЕИС, что создаёт потребность в альтернативных источниках.

**B2B-Center (b2b-center.ru)** — крупнейшая коммерческая ЭТП с **280 000+ компаний**. Уникальная особенность для строителей — выделенная площадка **B2B-Stroyka** (b2b-stroyka.ru) для строительных тендеров и закупок стройматериалов. **Единственная коммерческая площадка с полноценным API** для интеграции с ERP/SRM-системами (1С, Microsoft Axapta). Облачная SRM-платформа B2B Altis сертифицирована Минцифры как аналог SAP Ariba. Документация доступна только обладателям платных пакетов «СТАРтим».

**Тендер.Про (tender.pro)** выделяется возможностью участия **без ЭЦП** — уникальная характеристика среди российских площадок. Более 400 новых закупок ежедневно, 300 000+ поставщиков.

**Фабрикант (fabrikant.ru)** совмещён с госсекцией НЭП и обслуживает Росатом, Норильский Никель, ОСК. Предлагает страхование строительно-монтажных рисков прямо из личного кабинета.

### API и программный доступ к данным

С закрытием FTP-сервера ЕИС ландшафт доступа к данным кардинально изменился. Новый официальный канал — **SOAP-сервис**: `int44.zakupki.gov.ru/eis-integration/services/getDocsIP` (для физлиц, токен через Госуслуги) и `getDocsLE` (для юрлиц, требуется КЭП). Данные отдаются в XML-архивах. Доступные методы: `getDocsByReestrNumberRequest`, `getDocsByOrgRegionRequest`, `getNsiRequest`.

Развитая экосистема **коммерческих API** компенсирует ограничения официального канала:

- **ГосПлан API v2** (gosplan.info) — REST API в JSON, данные по 44-ФЗ, 223-ФЗ и ПП РФ №615. **Бесплатно до 01.07.2026**, далее 3 000 ₽/мес (600 запросов/мин). Извещения загружаются в течение 2 часов после публикации
- **DaMIA** (damia.ru) — REST API для поиска закупок, контрактов, поставщиков по ИНН/ОГРН
- **Мультитендер** (multitender.ru) — полная структурированная база 44-ФЗ, 223-ФЗ, ПП №615
- **NewDB** (newdb.net) — API для проверки контрактов по 44-ФЗ/223-ФЗ

### Юридический статус парсинга

Специального закона о парсинге в России нет. Общедоступная информация на ЕИС может свободно использоваться (ч. 2 ст. 65 44-ФЗ). Однако обход защитных механизмов (капча, авторизация) может квалифицироваться как несанкционированный доступ (ст. 272 УК РФ), а нарушение Terms of Service площадок — как гражданское правонарушение. С 2022 года часть закупок не публикуется в открытом доступе (ПП РФ №301 от 06.03.2022 — санкционная мера).

## 3. Документация к строительным тендерам: форматы и проблемы автоматизации

Строительная тендерная документация — самая объёмная среди всех категорий закупок. Крупные проекты содержат **100–500+ файлов общим объёмом от 1 до 10+ ГБ**.

### Форматы и состав

Типичный комплект включает: извещение (структурированная XML-форма в ЕИС), проект контракта (DOCX/PDF), техническое задание, проектную документацию (архивы ZIP/RAR с PDF и чертежами DWG), сметную документацию (XLSX, XML .gge, PDF), ведомость объёмов работ (XLSX/XML), заключение экспертизы (скан PDF).

**PDF остаётся доминирующим форматом**, причём значительная часть документации размещается в виде сканированных изображений — текст не извлекается, таблицы не парсятся. Это критическая проблема для автоматизации.

Позитивная тенденция: **XML-стандартизация развивается**. С августа 2021 года XML-формат (.gge) обязателен для локальных сметных расчётов на госэкспертизу, с января 2025 года — для ведомостей объёмов работ. Формат TIMML 1.02 введён для BIM-обмена (ПП РФ №614 от 17.05.2024). BIM-модели (IFC, RVT) обязательны для госпроектов с 2022 года (ПП РФ №331), но на практике в тендерной документации пока встречаются редко.

### Доступ и скачивание

Документация по 44-ФЗ доступна для скачивания **без регистрации** как на ЕИС, так и на соответствующих ЭТП. ФАС запрещает размещать документацию в защищённых паролем файлах. Для закупок по 223-ФЗ документация может быть размещена только на конкретной ЭТП.

Ключевое ограничение: **API ЕИС предоставляет только метаданные закупок (XML), но не прикреплённые файлы** (проектную документацию, сметы, чертежи). Для получения файлов необходимо обращаться к веб-интерфейсу ЕИС или ЭТП, что затрудняет массовое автоматическое скачивание.

### Возможности автоматического извлечения данных

| Параметр | Источник | Сложность извлечения |
|----------|----------|---------------------|
| НМЦК, сроки, место | Извещение (XML в ЕИС) | Низкая |
| Объёмы и виды работ | ВОР (XLSX/XML .gge) | Средняя–высокая |
| Требования к подрядчику | ТЗ/извещение | Средняя |
| Проектные решения, чертежи | PDF/DWG | Очень высокая |

Основные препятствия: сканированные PDF, отсутствие единых шаблонов, разнообразие форматов смет, нестабильность XML-схем ЕИС, огромные объёмы файлов.

## 4. Сметно-нормативная база: от ГЭСН к ресурсной модели

### Иерархия сметных нормативов

**ГЭСН (Государственные элементные сметные нормы)** — фундамент всей системы. Содержат количественные показатели затрат труда, машино-часов и материалов на единицу работ. Актуальная версия — ФСНБ-2022, в которой актуализировано **53 309 норм**. Разделены на строительные (ГЭСН), монтажные (ГЭСНм), ремонтные (ГЭСНр), пусконаладочные (ГЭСНп) и реставрационные (ГЭСНрр). Правообладатель — РФ в лице Минстроя, оператор — ФАУ «Главгосэкспертиза России».

**ФЕР (Федеральные единичные расценки)** — стоимостное выражение ГЭСН для базисного района (Московская область). **В ФСНБ-2022 ФЕР исключены** — их выпуск прекращён. Это ключевое изменение новой модели.

**ТЕР (Территориальные единичные расценки)** — адаптированы к условиям конкретного субъекта РФ с учётом районных коэффициентов, региональных цен на материалы и транспортных расходов. Разрабатываются региональными центрами ценообразования (РЦЦС). Обязательны для региональных и муниципальных бюджетных объектов.

**ОЕР (Отраслевые единичные расценки)** — для специализированных видов строительства: ОСНБЖ-2001 (РЖД), нефтехимические расценки (Роснефть, СИБУР, ЛУКОЙЛ), атомные расценки (Росатом). Разрабатываются отраслевыми ведомствами.

### Переход к ресурсно-индексному методу

**ФСНБ-2022** вступила в действие **25 февраля 2023 года** с базисом цен на 01.01.2022. Суть реформы — переход от базисно-индексного метода (БИМ) к **ресурсно-индексному (РИМ)**: каждый ресурс пересчитывается индивидуальным индексом или по текущей цене из ФГИС ЦС, вместо единого сводного индекса на итог сметы. Точность определения стоимости значительно возросла. С III квартала 2023 года все новые проекты обязаны использовать РИМ.

### Правовой статус и доступность

ФСНБ-2022 в формате XML **доступна для бесплатного скачивания** на портале ФГИС ЦС (раздел ФРСН). Это подтверждено письмом Минстроя №4350-ИТ/09 от 08.02.2021. Однако коммерческие сметные программы продают базы в проприетарных форматах с дополнительными сервисами — лицензия на базу в формате конкретной программы стоит **10 000–25 000 руб**.

## 5. ФГИС ЦС, индексы и мониторинг цен

**ФГИС ЦС (fgiscs.minstroyrf.ru)** — федеральная система ценообразования в строительстве, запущенная 30 сентября 2017 года. Оператор — Главгосэкспертиза, разработчик — АО «БАРС Груп». Система включает: Федеральный реестр сметных нормативов (ФРСН), Классификатор строительных ресурсов (КСР), подсистему мониторинга цен и расчёта индексов. Сметные цены на ценообразующие ресурсы и индексы обновляются **ежеквартально**. Поставщики стройматериалов с оборотом от 10 млн руб./год обязаны подавать данные.

**Полноценного REST API у ФГИС ЦС нет.** Данные публикуются в XML и XLS. Сметные программы загружают цены и КСР через встроенные механизмы. Для поставщиков работает интеграция через «1С:Передача данных в ФГИС ЦС».

**Индексы пересчёта** публикуются ежеквартально письмами Минстроя для каждого субъекта РФ в разрезе видов строительства и элементов затрат. Публикуются на сайте Минстроя, в ФГИС ЦС и справочных системах (КонсультантПлюс, Гарант). Региональные коэффициенты включают: территориальные индексы пересчёта, районные коэффициенты к зарплате (**от 1,15 до 2,0**), коэффициенты зимнего удорожания (8 температурных зон), территориальные поправки к ФЕР. Некоторые субъекты делятся на несколько ценовых зон (ХМАО-Югра — 5 зон).

### Мониторинг цен на стройматериалы

- **Стройинформресурс** (stroycena.online) — крупнейший коммерческий разработчик каталогов текущих цен (КТЦ), ежемесячные выпуски для всех регионов. Данные поставляются в форматах сметных программ
- **НОСТРОЙ** (nostroy.ru) — еженедельный мониторинг цен с интерактивной картой поставщиков, публичные данные без API
- **СтройСканер** (stroyscan.ru) — мониторинг цен на металлопрокат, бетон и стройматериалы с онлайн-калькулятором
- **Синтека** (cynteka.ru) — платформа автоматизации закупок для строительных компаний
- **fgisrf.ru** — единственный обнаруженный коммерческий **API для доступа к расценкам ФСНБ-2020 и ФСНБ-2022**

## 6. Программы для составления смет: от Гранд-Сметы до облачных сервисов

**ГРАНД-Смета (grandsmeta.ru)** — безусловный лидер с долей **~75% рынка** и более 150 000 пользователей. Версия «Проф» — от **30 000 руб.**, «Флеш» — 35 000 руб. Поддерживает все методы расчёта, ФСНБ-2022, экспорт в XML для экспертизы. Обмен через собственный формат, XML и АРПС 1.10. Публичного REST API нет.

**Smeta.RU (smeta.ru)** — второй по популярности, более 30 000 пользователей, лидер в Москве. От **15 000 руб.** для физлиц, от 25 000 руб. для организаций. Облачная версия Smeta.Cloud работает по подписке без покупки нормативных баз. Обмен через АРПС, БОСБ, XML. Интеграция с 1С. **АтомСмета** — отраслевая версия для «Росатома».

**1С:Смета** (solutions.1c.ru) — от **7 200 руб.** Ключевое преимущество — глубокая интеграция в экосистему 1С: ERP, Подрядчик строительства, Бухгалтерия, BIM 6D. Загрузка КСР и цен из ФГИС ЦС. Машинное обучение для заполнения реквизитов. Платформа 1С:Предприятие 8 предоставляет REST API для разработчиков.

**ПК РИК (smetarik.ru)** — от **39 900 руб.**, более 40 000 клиентов, используется РЦЦС для разработки ТЕР и индексов. Лидер в Калининградской области.

**Турбосметчик (turbosmeta.ru)** — от **30 000 руб.**, стандарт для ОАО «РЖД» (~70% структур). Уникальная технология ВизАРС с цветовой маркировкой ошибок.

**Комплекс А0 (gk-infostroy.ru)** — лидер в Санкт-Петербурге, корпоративная MS SQL-версия на 100+ рабочих мест. **Единственная из основных программ с заявленным API** для интеграции в корпоративные системы. Обмен с Oracle Primavera.

Стандартный формат межпрограммного обмена — **АРПС 1.10** (обязателен в Москве по распоряжению №740-РП). Все программы поддерживают XML-экспорт для Главгосэкспертизы.

Среди облачных решений нового поколения выделяются **Gectaro** (gectaro.com) — сметы, бюджеты, закупки в одном сервисе с импортом из ГРАНД-Сметы и Smeta.RU; **ПростоСмета** (простосмета.рф) — бесплатная облачная программа для коммерческих расценок; **Адепт:Смета** — расчёт на основе BIM-моделей с интеграцией nanoCAD, Renga, Revit, IFC.

## 7. Сервисы мониторинга тендеров: кто есть кто

Рынок аналитических сервисов динамично развивается. Ключевые игроки отличаются охватом площадок, глубиной аналитики и наличием AI-функций.

**Контур.Закупки** (zakupki.kontur.ru) — один из лидеров рынка, **440+ площадок**, ML-фильтрация по отраслям, API для CRM (1С, Битрикс24, amoCRM), Telegram-уведомления. Тарифы: Стандарт — **23 600 руб./год**, Эксперт — 46 400 руб./год.

**Seldon.Pro** (seldon.pro) — единственный массовый сервис с полноценным **AI-прогнозом вероятности победы в %**, прогнозом конкурентного окружения, обоснованием НМЦК (Seldon.Price), проверкой контрагентов (Seldon.Basis). Тарифы от ~15 000 до **60 000+ руб./год**, индивидуальное ценообразование. Поддерживает ПП РФ №615 (капремонт).

**Saby Trade** (saby.ru/tenders) — рекордный охват в **700+ площадок**, включая международные, поиск в сканах и архивах, мобильное приложение, API. Тарифы: 15 000–34 000 руб./год. Экосистема СБИС.

**Закупки360** (zakupki360.ru) — лучшее **бесплатное решение** с качественным поиском и прогнозом конкурентов. Платные тарифы: 12 650–18 480 руб./год.

**Bicotender** (bicotender.ru) — старейший сервис (с 2005 г.), **6 000+ источников**, семантический поиск, TRM (тендерная CRM), Telegram-бот. Тарифы: 22 900–59 900 руб./год.

**Тендерплан** (tenderplan.ru) — хороший баланс цена/функционал, интеграция с Битрикс24. От **8 900 руб./3 мес.** TenderGuru (tenderguru.ru) заявляет **56 000+ источников** — формально крупнейшую базу.

### AI-функции на рынке

Рынок AI в тендерах находится на **ранней стадии**. По оценке «Цифровых Закупок», полноценный AI-функционал не предлагает ни один массовый сервис — большинство работают на традиционных алгоритмах. Seldon.Pro ближе всех к полноценному AI (прогноз победы). Новые стартапы активно внедряют NLP/LLM: **ZakupAI** (zakupai.online) — AI-прогноз победы и анализ ТЗ через Telegram-бот; **ТендерИнтеллект** (tender-intellekt.ru) — нейросетевой поиск на естественном языке; **Zakupki Assistant** — AI-анализ PDF без чтения (сроки, обеспечение, риски), автоопределение ОКПД2; **Tender Sniper** — AI-модель оценки соответствия тендера бизнесу.

Внутренний кейс: девелопер «Самолёт» тестировал нейросеть для проверки смет по 6 000+ источникам и обнаружил переплату в **28 млн руб.**, но это закрытое решение.

## Заключение: три незакрытые ниши и оптимальная архитектура

Исследование выявляет три ключевых разрыва на рынке. Во-первых, **автоматическая генерация сметных прототипов** на основе тендерной документации — абсолютно незанятая ниша; ни один публичный сервис этого не предлагает. Во-вторых, **глубокий анализ строительной специфики** (виды СМР, допуски СРО, объёмы работ) не реализован ни в одном тендерном сервисе — все работают на уровне базовых фильтров. В-третьих, отсутствуют интегрированные решения, связывающие мониторинг тендеров, анализ документации и сметный расчёт в единый pipeline.

Оптимальная техническая архитектура для входа в этот рынок: **ГосПлан API** (бесплатный до июля 2026) или SOAP API ЕИС для метаданных → веб-скрапинг ЕИС/ЭТП для файлов документации → OCR для сканированных PDF + парсинг XLSX-смет + обработка XML .gge → формат АРПС 1.10 для интеграции со сметными программами → данные ФГИС ЦС и fgisrf.ru (коммерческий API) для текущих цен. Строительный сегмент — крупнейший и одновременно наименее автоматизированный, что создаёт существенное конкурентное окно.
